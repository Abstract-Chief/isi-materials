## Кластеризация

K-Means: делит данные на K кластеров.  
Elbow method для выбора K.  
Distortion - сумма квадратов расстояний точек до центров.

## Кросс валидация
Разделение на K частей, каждая становится тестовой один раз. Даёт стабильную оценку качества.

## Выборка данных

VarianceThreshold — удаляет колонки без изменчивости.  
RandomForest + SelectFromModel — выбор важных признаков.  
RFECV — рекурсивное удаление с кросс-валидацией.

## Алгоритмы Number Puzzle

### BFS
Поиск в ширину.

### DFS
Уходит в глубину пока не упрётся или не превысит лимит.

### Greedy search
Выбирает ближайший к цели шаг по эвристике.

### A*
Greedy + учитывает расстояние от начальной точки.

## Судоку

Домен — множество возможных значений клетки.  
FC — удаляет невозможные значения, откатывается если пусто.  
MRV — выбираем клетку с минимальным количеством вариантов.  
LCV — выбираем значение, ограничивающее соседей меньше всего.
